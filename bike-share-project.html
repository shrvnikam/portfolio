Bike-Share Analysis
This project analyzes the usage of Cyclistic bikes, comparing the behavior of annual members and casual riders to drive recommendations for increasing revenue. The analysis focuses on identifying patterns in ride durations, ride types, and the time of year and day when bikes are used.

Code for Analysis

1. Data Import and Cleaning

# Importing libraries
import pandas as pd
import numpy as np

# Reading data files
data = pd.read_csv('path_to_file.csv')

# Data cleaning
data['ride_duration'] = pd.to_numeric(data['ride_duration'], errors='coerce')
data.dropna(subset=['ride_duration'], inplace=True)

2. Ride Duration Analysis by User Type

# Ride duration analysis by user type
ride_duration_analysis = data.groupby('user_type')['ride_duration'].mean()

# Plotting average ride duration by user type
import matplotlib.pyplot as plt
ride_duration_analysis.plot(kind='bar')
plt.title('Average Ride Duration by User Type')
plt.xlabel('User Type')
plt.ylabel('Average Ride Duration (mins)')
plt.show()

3. Heatmap Analysis by Day of the Week and User Type

# Heatmap of rides by day of the week and user type
import seaborn as sns

pivot_data = data.pivot_table(values='ride_id', index='day_of_week', columns='user_type', aggfunc='count')

plt.figure(figsize=(10,7))
sns.heatmap(pivot_data, annot=True, cmap="YlGnBu")
plt.title('Heatmap of Rides by Day of the Week and User Type')
plt.show()

4. Proportional Analysis of Ride Types

# Ride types analysis
ride_type_analysis = data.groupby('user_type')['ride_type'].value_counts(normalize=True).unstack()

# Plotting proportional analysis
ride_type_analysis.plot(kind='bar', stacked=True)
plt.title('Proportional Analysis of Ride Types by User Type')
plt.ylabel('Proportion')
plt.show()

5. Trend Analysis Over Time

# Trend analysis over time
data['ride_date'] = pd.to_datetime(data['ride_date'])

monthly_trend = data.groupby(data['ride_date'].dt.to_period('M'))['ride_id'].count()

# Plotting trends over time
monthly_trend.plot(kind='line')
plt.title('Trends Over Time')
plt.xlabel('Month')
plt.ylabel('Total Rides')
plt.show()
